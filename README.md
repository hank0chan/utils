# utils

一个简单的个人工具类项目，主要提供了一些基于原有JDK的封装工具类，提高开发效率。

## 使用

引入以下maven依赖即可使用：
```
    <dependency>
        <groupId>cn.hankchan</groupId>
    	<artifactId>utils</artifactId>
    	<version>${branch}</version>
    </dependency>
    
```

其中，版本号${branch}对应项目地址的可读分支，如：0.0.1, 0.0.2等。
